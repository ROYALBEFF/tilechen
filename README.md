# Tilechen 🍰

Tilechen is a tile viewer and editor for GameBoy, GameBoy Color and GameBoy Advance ROMs.

## Feature completion

| System          | View | Edit |
|-----------------|------|------|
| GameBoy         | ✅   | ❌    |
| GameBoy Color   | ✅   | ❌    |
| GameBoy Advance | ❌   | ❌    |

## Installation
Install as `uv` tool:
```
uv tool install tilechen
```

Install from PyPi:
```
pip install tilechen
```
